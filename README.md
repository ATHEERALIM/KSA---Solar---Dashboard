KSA SOLAR DASHBOARD
Project Overview
This project is an Interactive Data Dashboard built with Python to analyze solar radiation efficiency across 5 major cities in Saudi Arabia (Riyadh, Jeddah, Mecca, Medina, and Dammam) for the year 2024. The application transforms raw environmental data into actionable insights, supporting the Saudi Vision 2030 goals for renewable energy expansion.
 Tech Stack
• Language: Python 3.x
• Web Framework: Streamlit
• Data Manipulation: Pandas
• Visualization: Plotly (Interactive Charts)
• Deployment: GitHub & Streamlit Community Cloud
 Key Features
• Multi-City Comparison: Compare solar radiation (GHI) and temperature trends between different regions.
• Dynamic Filtering: Users can filter data by city and customize chart color themes in real-time.
• Geographical Analysis: Insights into solar efficiency based on Latitude and Longitude coordinates.
• Interactive Tooltips: Instant data display and zooming capabilities for precise analysis.
 Project Structure
• app.py: The main Python script containing the dashboard logic.
• requirements.txt: List of necessary Python libraries (Streamlit, Pandas, Plotly, Matplotlib).
• ksa_solar_dataset_2024.csv: The dataset containing solar and weather records.
